  • Γενικά

  • Είμαι γονιός, πως εγγράφομαι

    Απλά πατάς εγγραφή απο την μπάρα στην κορυφή σελίδας. Συμπληρώνεις τα στοιχεία σου, πρώτα προσωπικές πληροφορίες και ύστερα τα στοιχεία του λογαριασμού σου, και γίνεσαι μέλος της οικογένειας του Kiddom.

  • Είμαι πάροχος, πώς εγγράφομαι ;

    Διαλέγεις τη επιλογή "Πάροχοι" και απο εκεί μπορείς να συμπηρώσεις τα στοιχεία σου. Με δύο απλά βήματα, ενα για τα προσωπικά σου στοιχεία και ένα για τα στοιχεία του λογαριασμού σου. Αφού ένας admin μας εγκρίνει την εγγραφή σου είσαι έτοιμος να προσφέρεις διασκεδαστικές εμπειρίες σε νέους και νέες.

  • Μπορώ να αλλάξω τα προσωπικά μου στοιχεία

    Αφου εγγραφείς μπορείς να πλοηγηθείς στο προφίλ σου μέσω της μπάρας στην κορυφή της σελίδας. Εκεί μπορείς να πλοηγηθείς στις προσωπικές σου πληροφορίες αλλα και να αλλάξεις τα στοιχεία σου.

  • Σε τι ηλικίες απευθύνεται το Kiddom

    Το Kiddom είναι ένας ιστοχώρος που έχει στόχο να προσφέρει χαρά και γνώση σε παιδιά κάθε ηλικίας, ακόμα και αν είναι γονείς.

  • Πολιτική κράτησης δραστηριότητας

  • Πώς γίνεται η κράτηση

    Από τη σελίδα της κάθε δραστηριότητας. Εκεί μπορείς να διαλέξεις την ημερομηνία και τη ώρα που σε ενδιαφέρει για την δραστηριότητα. Πατάς κράτηση και χρεώνεσαι τους αντίστοιχους πόντους

  • Μπορώ να κρατήσω πολλές θέσεις σε μια δραστηριότητα

    Ναι! Κατά την επιλογή ημέρας και ώρα για τη δραστηριότητα προσφέρεται η δυνατότητα να κρατήσεις πολλές θέσεις.

  • Μπορώ να ακυρώσω τις κρατήσεις μου;

    Έχεις δικαίωμα να ακυρώσεις κάποια απο τις κρατήσεις σου εντός διαστήματος μέχρι 36 ωρες πριν την διεξαγωγή της δραστηρίοτητας.

  • Πολιτική πληρωμών

  • Πως πληρώνω;

    Το Kiddom, ακολουθεί μια πολιτική πληρωμής με πόντους. Έχεις τη δυνατότητα να αγοράσεις πόντους και στη συνέχεια να πληρώσεις με αυτούς για μια θέση σε μια δραστηριότητα. Οι πόντοι δεσμεύονται μέχρι και 36 ώρες πριν την δραστηριότητα, δηλαδή όσο είναι δυνατό να γίνει ακύρωση, και έπειτα αφαιρόυνται.

  • Αν ακυρώσω μου επιστρέφονται οι πόντοι ;

    Σε περίπτωση ακύρωσης οι πόντοι επιστρέφονται αρκεί η ακύρωση να γίνει περισσότερο απο 36 ώρες πριν την εναρξη της δραστηριότητας. Σε περίπτωση ακύρωσης σε διάστημα λιγότερο των 36 ωρών απο την ώρα διεξαγωγής της δραστηριότητας, οι πόντοι αφαιρούνται.

  • Πως μπορώ να δω τους πόντους μου;

    Στη σελίδα του Προφίλ, υπάρχει η επιλογή "Πορτοφόλι" στην οποία δίνεται μια αναφορά για την κατάσταση των πόντων σου. Μπορείς να δεσμευμένους πόντους σε κρατήσεις, πόντους που έχεις πληρώσει και φυσικά τους διαθέσιμους πόντους σου.

  • Πώς αγοράζω πόντους;
    Στο "Πορτοφόλι" στη σελίδα του προφίλ υπάρχει η επιλογή για αγορά πόντων. Στη συνέχεια απλά επιλέγεις τα πακέτα πόντων που επιθυμείς. Εισάγεις τα στοιχεία της πιστωτικής σου κάρτας και ολοκληρώνεις τη συναλλαγή σου.
  • Είμαι πάροχος, πότε θα πληρωθώ ;

    Ως πάροχος υπηρεσιών θα πληρώνεσαι στην αρχή κάθε μήνα το σύνολο των πόντων που σου αναλογούν μετατρεμμένο σε ευρώ.
